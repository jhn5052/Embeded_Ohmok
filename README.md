# Embeded_Ohmok

### Team
> 최종윤, 한민지

### Description
> S3C2450 보드에서 오목 게임을 구현

### Tools
>  - Host PC: windows 10 OS
>  - Target Board: S3C2450(ARM CORE)
>  - Builder : GNU-GCC
>  - Virtual Linux for Build: Cygwin
>  - Editor(IDE) : DEV C++
>  - Tools for DebugL Tera Term
>  - Image Convertor : 24bit BMP to 16Bit BMS


### Tech 
>  - Embeded-C
>  - Cross Compiler
>  - LCD 
>  - Touch Screen
>  - Timer
>  - Interrupt(Touch_ISR, Timer_ISR)
>  - Buzzer
